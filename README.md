🧠 What is PromptCraft?

PromptCraft is an educational tool that helps users bridge the gap between "asking a question" and "engineering a result." It utilizes the C.R.E.A.T.E framework to help users build complex, high-accuracy instructions for Large Language Models (LLMs).

Key Features

The Anatomy Explorer: An interactive breakdown of the five core pillars of a professional prompt.

Performance Analytics: Real-time charting (via Chart.js) demonstrating the accuracy gains of structured prompting over raw "zero-shot" queries.

AI Prompt Optimizer: A local heuristic engine that critiques user prompts and provides an optimized alternative instantly.

Interactive Prompt Builder: A "code-window" style interface that allows users to toggle architectural layers (Role, Context, Constraints) to compile a master prompt in real-time.

🛠️ Tech Stack

Frontend: HTML5, Tailwind CSS (via CDN)

Charts: Chart.js (Canvas-based rendering)

Logic: Vanilla JavaScript (ES6+)

Typography: Inter & JetBrains Mono (via Google Fonts)

📂 Project Structure

This project is built as a Single File SPA (Single Page Application).

index.html: Contains all HTML structure, Tailwind styling, and JavaScript logic. This makes the project extremely portable and easy to host on services like GitHub Pages or Netlify.
